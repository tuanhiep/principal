# Principal

Engineering leadership and Staff+ growth blog.

## Purpose

This repository hosts long-form writing about engineering leadership, Staff+ thinking, software architecture, and career growth.

## Contents

- Blog posts under `content/`.
- Static assets under `static/`.
- Hugo layouts, theme configuration, and generated public output.

## How To Use

Edit source content, preview locally with Hugo, and publish through the configured deployment flow.
